Upload tài liệu trực tiếp lên web

Trang này dành riêng cho bạn. Sau khi đăng nhập, mình có thể tải PDF, hồ sơ năng lực, bảng giá hoặc tài liệu kỹ thuật lên Supabase Storage và lấy link ngay để gửi khách.

Mình sẽ dùng để làm gì?

Upload file, lấy link public, copy nhanh và mở kiểm tra trực tiếp trên website.

Lưu ý

Bucket Supabase nên để public nếu bạn muốn link tài liệu mở trực tiếp cho khách.

Đăng nhập quản trị

Nhập mật khẩu riêng của bạn để mở khu upload tài liệu.

Upload tài liệu

Chọn file từ máy tính rồi bấm upload. File mới sẽ hiện ngay trong danh sách bên phải.

Gợi ý: phù hợp nhất với PDF, DOCX, XLSX, ảnh công trình, hồ sơ năng lực và bảng báo giá. Giới hạn hiện tại: 25MB/file.
